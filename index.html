Giới thiệu

Xin chào! Mình là Trần Huy Vui — một backend developer chuyên xây dựng các API hiệu năng cao và kiến trúc phân tán. Mình đặc biệt quan tâm đến thiết kế hệ thống cơ sở dữ liệu Hybrid (SQL + NoSQL), tối ưu hiệu suất với Redis caching, và ứng dụng AI vào các bài toán thực tế.

Mình có nền tảng vững về Node.js, TypeScript, Express ở phía backend và có thể làm việc hiệu quả với Vue 3, Tailwind CSS ở phía frontend khi cần. Luôn hướng đến code sạch, có cấu trúc, và dễ scale.
